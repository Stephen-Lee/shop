在线书店
==============
一个简易商店系统
功能：

商品展示,购物车,收藏,评论,搜索

后台: 商品管理， 多级分类管理，用户(管理员)管理，订单管理。
用户分布、年龄、性别图表统计，营业额统计。

获得seed data：

    rake db:populate
